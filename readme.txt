simple local host webpage and rest client for data source
  

to run:
1- local env- will need to start djangoRestApi (python rest host) and postgres server prior. 
2- app.  manual: in term node [filename].js  plus enter key  file = index.js mainly.  typical main entry point.
  auto/ debug: see Run & Debug bar to left & click arrow
3- in browser type http://localhost:3000

instantitation:
after dir created, in terminal type: npm init 
  creates package.json

source:
Rest client portion: https://reintech.io/blog/how-to-use-node-js-to-create-a-rest-client

-axios
-express
